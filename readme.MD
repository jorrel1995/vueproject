This is a project to learn Laravel and Inertia.js with Vue 3. The project is a simple CRUD application for managing users with admin and user roles. 

## Features

- Admin and user roles
- User management
- Role management
- Permission management

- Password: password

To run the project:

1. Clone the repository
2. Run `composer install`
3. Run `npm install`
4. Run `php artisan migrate`
5. Run `php artisan db:seed`
6. Run `php artisan serve`
7. Run `npm run dev`
